Mon, Jan 1, 0001

(Corresponds to commit 4bd8422107 in earlywarning repo) For the individual-based simulation, the population dynamics are given by $$\begin{align} \frac{dP(n,t)}{dt} &= b{n-1} P(n-1,t) + d{n+1}P(n+1,t) - (b_n+d_n) P(n,t), b_n &= \frac{e K n^2}{n^2 + h^2}, d_n &= e n + a, \end{align}$$ which is provided by the saddle_node_ibm model in populationdynamics. For each of the warning signal statistics in question, we need to generate the distibution over all replicates and then over replicates which have been selected conditional on having experienced a crash.
