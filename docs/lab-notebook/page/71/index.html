Implementing an example of low and high power trees

Mon, Jan 1, 0001

Code implementation in felsenstein_tree.R I consider a star tree and a Felsenstein tree each of N nodes; M = (N+1)/2 tips. I consider an uncorrelated data set of M normal random variables, such as would be generated by either BM or OU models on the star tree. I produce a correlated data set such as would be generated by Brownian motion on the felsenstein tree. I generate 8 model fits – fitting BM and OU to each tree under each data set.
