Comparing to Analytic Dynamics

Tue, Feb 23, 2010

Goal for today: implement analytic calculations for warning signals. 10am-12pm Abstraction/functionalizing code. 4pm-7pm Implement GSL ode solvers for linear noise approximation: Recall our dynamics are d(n) = *e*n + a and defining the jump moments (van Kampen) α~1~(n) = b(n) − d(n) α~2~(n) = b(n) + d(n)

recall that we have a linear Fokker-Planck equation whose solution is Gaussian and specified by the first two moments: These are implemented in code using GSL ode library.
