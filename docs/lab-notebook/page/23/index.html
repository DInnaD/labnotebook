Tue, Dec 10, 2013

Consider the model $$d X_t = \alpha \left(\theta - X_t\right)dt + \sigma dB_t $$ By Ito Isometry, we have: $$\langle X \rangle_t = E_t(X) = \theta \left(1 - e^{-\alpha t} \right) + X_0 e^{-\alpha t}$$ and $$\langle X^2 \rangle_t - \langle X \rangle_t^2 = Vt(X) = \frac{\sigma^2}{2 \alpha }\left(1 - e^{-2 \alpha t} \right)$$ If we assume discrete, uniform sampling of spacing $\tau$, then we have, $$P(\theta | X; \alpha, \sigma) = \sqrt{\frac{1}{2\pi V{\tau} }}^{T-1} \exp\left(-\frac{\sumt^{T-1} \left(X{t+1} - E\right)^2 }{2V}\right) $$
