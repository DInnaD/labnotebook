Sat, Sep 7, 2013

We have also recently reached the first milestone in RNeXML. We can now read NeXML files into R and convert to the popular phylo format from the ape package, as well as writing the ape::phylo trees out into valid NeXML. We can also add basic metadata such as author, license, title and description to the file. Development notes At the heart of this package is an internal S4 representation of the schema.
