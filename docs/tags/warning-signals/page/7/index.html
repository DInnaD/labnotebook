Crashing stocks and optimal harvesting effort on Allee effects

Wed, Nov 9, 2011

Modeling choices Allee effect model: Myers Beverton-Holt with harvesting-effort mortality: $$ x_{t+1} = f(x) = \frac{a x_t^{\alpha}}{1+ x_t^{\alpha}/b} - e x $$ For $ \alpha = 2 $, equilibrium points are $ x = 0 $ and $$ x = \frac{ab}{2(1+e)} \pm \frac{1}{2} \sqrt{ \left(\frac{ab}{1+e}\right)^2-4b } $$ As written this differs from our earlier solutions a couple respects – the growth function applies to the unharvested population, which then experiences harvesting mortality, and the control variable is fishing effort, not harvest.

Friday: Optimal Control

Fri, Oct 21, 2011

Optimal control Imagine we’re trying to minimize $J = \theta( y(t_F), t_F)$ subject to both the state equations: $\dot y = f(y(t), u(t))$, Note these are continuous, applying at all times $t$. and boundary conditions: $\psi(y(t_F), u(t_F), t_F) = 0$. These are discrete, applying only at fixed time $tF $. We have to put the continuous constraint in as an integral: $$ \hat J = \left[\phi + \mathbf{\nu}^T \psi \right]{tF} - \int{t_I}^{t_F} \mathbf{\lambda}^T \left(\dot{\mathbf{y}} - f(\mathbf{y}(t),\mathbf{u}(t)) \right) dt $$
